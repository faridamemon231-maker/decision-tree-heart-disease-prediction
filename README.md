# 🩺 Decision Tree for Heart Disease Prediction

## 📌 Overview

This project uses a Decision Tree Classifier to predict the presence of heart disease based on patient health data. The objective is to analyze medical attributes and build a machine learning model capable of classifying whether a patient is likely to have heart disease.

---

## 🎯 Objectives

- Analyze heart disease data.
- Perform data cleaning and preprocessing.
- Detect and handle outliers.
- Train a Decision Tree Classification model.
- Evaluate model performance.
- Optimize model accuracy using tree depth tuning.

---

## 📂 Dataset Features

The dataset contains the following medical attributes:

- Age
- Sex
- Chest Pain Type (cp)
- Resting Blood Pressure (trestbps)
- Cholesterol Level (chol)
- Fasting Blood Sugar (fbs)
- Resting ECG (restecg)
- Maximum Heart Rate (thalach)
- Exercise Induced Angina (exang)
- Oldpeak
- Slope
- CA (Coronary Artery)
- Thal

### Target Variable

- 1 = Heart Disease Present
- 0 = No Heart Disease
